METALFUSION WEBSITE - STARTUP GUIDE
===================================

PREREQUISITE: Node.js must be installed (https://nodejs.org)

DEVELOPMENT MODE (local preview):
  1. Open Terminal
  2. Navigate to the folder: cd metalfusion-website
  3. Run: npm install
  4. Run: npm run dev
  5. Open: http://localhost:5173

PRODUCTION BUILD:
  npm run build
  (output: dist/ folder - upload this to the hosting provider)

PAGES:
  /                       -> Home
  /rolunk                 -> About
  /szolgaltatasok         -> Services
  /szolgaltatasok/[slug]  -> Individual service
  /referenciak            -> References list
  /referenciak/[slug]     -> Individual project
  /galeria                -> Image gallery (lightbox)
  /ajanlatkeres           -> Quote request form
  /kapcsolat              -> Contact
  /adatvedelem            -> Privacy policy
  /404                    -> Error page

TECHNOLOGY: Vite + React 18 + React Router v6
